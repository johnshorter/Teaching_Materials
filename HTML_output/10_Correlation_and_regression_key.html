These tables are not in tidy format. The years spread out across 200+ columns. Let’s make them into the tidy format first, and then we’ll join them by matching the country and year columns.

child_mortality_tidy <- child_mortality %>% 
  pivot_longer(names_to = "year", values_to = "death_per_1000", cols = c(2:302)) 

income_tidy <- income %>% 
  pivot_longer(names_to = "year", values_to = "income", cols = c(2:242)) 

babies_per_woman_tidy <- babies_per_woman %>% 
  pivot_longer(names_to = "year", values_to = "birth", cols = c(2:302)) 

mortality_and_income_and_birth <- child_mortality_tidy %>% 
  inner_join(income_tidy, by = c("country", "year")) %>%
  inner_join(babies_per_woman_tidy, by = c("country","year")) %>% 
  mutate(log10_income = log10(income))

Curve fitting using linear model

A limitation of the simple linear regression is that a linear relationship is required. However, under certain circumstances, you can also fit curves using linear model.

You can fit curves using linear model when the underlying mathematical relationship is given.

To understand that, let’s use the R built-in dataset Puromycin as an example. In this experiment, we are looking at the rate of an enzyme across different substrate concentrations. There are two states: drug treated or non-treated. We will only use the non-treated control data for now. (Ref: Treloar, M. A. (1974), Effects of Puromycin on Galactosyltransferase in Golgi Membranes, M.Sc. Thesis, U. of Toronto.)

Puromycin_ctrl <- Puromycin %>% 
  filter(state == "untreated")

head(Puromycin_ctrl)
##   conc rate     state
## 1 0.02   67 untreated
## 2 0.02   51 untreated
## 3 0.06   84 untreated
## 4 0.06   86 untreated
## 5 0.11   98 untreated
## 6 0.11  115 untreated

1/V = 2.15e-4*(1/S) + 6.972e-3

So Vmax = 1/6.972e-3, and K = 2.15e-4/6.972e-3

1/6.972e-3 #Vmax
## [1] 143.4309
2.15e-4/6.972e-3 #K
## [1] 0.03083764

So Vmax = 143, and K = 0.03

In addition, we have an R^2 = 0.89, which means 89% of the variation in the data are explain by the model. Pretty good.

Exercise two

Now let’s practice linear regression. We’ll use the Puromycin data again. However, we will use the drug treated data this time. Again, we are looking at the rate of an enzyme across substrate concentrations, but this time the enzyme is treated with the drug puromycin.

Puromycin_treat <- Puromycin %>% 
  filter(state == "treated") %>% 
  mutate(one_over_v = 1/rate) %>% 
  mutate(one_over_s = 1/conc)

fitted_rate_puro <- data.frame(
  "one_over_s" = seq(min(Puromycin_treat$one_over_s), max(Puromycin_treat$one_over_s), by = 0.1)
) %>%
  mutate(one_over_v = 0.000247*(one_over_s) + 0.0051) %>% 
  mutate(conc = 1/one_over_s) %>% 
  mutate(rate = 1/one_over_v)
